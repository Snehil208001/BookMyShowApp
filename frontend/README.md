# BookMyShow Frontend

React frontend for the BookMyShow movie booking service.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Runs at http://localhost:5173

**Ensure the backend is running at http://localhost:8080**

## Flow

1. **Home** - Browse movies, search
2. **Movie** - Select venue and showtime
3. **Seats** - Select seats → Reserve (10 min) → Confirm booking
4. **Orders** - View booking history

## Build

```bash
npm run build
```
